# Bank Marketing Prediction using Decision Tree Classifier

## Project Overview

This project uses a **Decision Tree Classifier** to predict whether a customer will purchase a product or subscribe to a service based on demographic and behavioral data.
The project is built using the **Bank Marketing Dataset** from the UCI Machine Learning Repository.

## Dataset
Dataset used:
Bank Marketing Dataset from UCI Repository
- Dataset contains customer information collected during marketing campaigns.
- Goal is to predict whether the customer will subscribe to a term deposit.

Target Variable:
- `y`
  - yes = customer subscribed
  - no = customer did not subscribe
Dataset Link:
https://archive.ics.uci.edu/ml/datasets/bank+marketing

## Technologies Used

- Python

## Machine Learning Algorithm

### Decision Tree Classifier

A Decision Tree Classifier is a supervised machine learning algorithm used for classification tasks. It splits the dataset into branches based on feature conditions to make predictions.
